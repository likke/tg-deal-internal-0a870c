The difference is depth and time: SG's serious tiers run tens-to-hundreds of self-paced hours; AI Ready is intensive, in-person, 3-day. They're complementary, not identical — so claim AI4K/AI4S parity honestly, and don't claim AI4I-equivalence at college.
03 /Gaps to close before they share their syllabus
  1. Depth at the college tier. A 3-day "AI + Automation" won't stand next to AI4I's 140 hours of Python/ML. Done → college repositioned as applied automation (no-code + light Python), and the optional AI Ready Advanced tier (₱5,000, blended + self-paced) now covers AI4I-level depth.
  2. Assessed, stackable certification. SG certs are assessment-gated and "industry-recognized." A certificate of attendance won't carry the "national standard" claim. Done → the Assessment Framework is built — assessed at every level (build + knowledge check + presentation), a pass rubric, and a verifiable cert ID + registry. Production step left: quiz banks + the /verify page.
  3. A visible progression pathway. SG is a ladder a learner climbs over years; AI Ready currently reads as 3 separate cohorts. Done → the pathway is now on the program page: Awareness → Literacy → Automation → Advanced, with the credential stacking at each step.
  4. Educator resource pack. SG hands schools reusable, educator-led classroom materials. Done → the Educator Resource Pack is specced — 10-part trainer bundle (facilitator guide, lesson plans, decks, workbooks, project briefs, assessment kit, certificates, safeguarding, setup). Production step left: produce pilot-level materials.
04 /How to position it — and one thing to know
"AI Ready is modeled on Singapore's national AI literacy ladder — AI4K → AI4S → AI4I — delivered as an intensive, in-person, project-based program built for Filipino classrooms, with stackable assessed certification and a certified-trainer network."

That claim is true and defensible on structure. It positions you as the SG-equivalent IP provider for the AI track without overclaiming on depth.

Know this before the NDA exchange: AI Singapore already runs AI4K Train-the-Trainer in the Philippines (and Malaysia & Thailand). That's both validation (the model works here) and a signal — an SG-origin program is already touching PH schools. Worth confirming whether Think Global's "Singapore partner" is in this orbit, because their syllabus may map directly to AI4K/AI4S.
